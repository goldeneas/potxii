# Progetto PotXII

## Interfaccia
- Dashboard NodeRED
- MQTT/Wifi
- Diagramma di linea per ogni grandezza misurata
- Grafico per indicare il livello dell'acqua nel tempo
- Log per eventi eccezionali
- Soglia umidita' per irrigazione
- Livello luminosita' dei LED regolabile
- Attivazione manuale pompa

## Sensori/Componenti Utilizzati
- Pompa ad immersione con rele'
- HCSR04
- Sensore per umidita' del terreno
- TSL2561
- SSD1306
- DHT
- Power supply esterno
- LED con PWM
- Pulsante per reset
- MB102

## Features
- Start at boot
- Notifica dashboard serbatoio vuoto
- Misurazione umidità terreno ed aria, luminosità e temperatura aria
- Analisi delle grandezze misurate
- Irrigazione automatica
- Illuminazione con LED
